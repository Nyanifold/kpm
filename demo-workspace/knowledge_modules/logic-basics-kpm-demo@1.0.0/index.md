# Logic Basics

Welcome to **Logic Basics** — a structured introduction to mathematical logic, the formal language underlying all rigorous reasoning in mathematics, computer science, and philosophy.

Logic studies what it means for an argument to be valid. Rather than asking whether premises are true in the world, formal logic asks: *if* the premises are true, *must* the conclusion follow? This separation of form from content makes logic universally applicable.
