# algebra-basics

A knowledge package covering the foundations of algebra, from set notation through linear equations. No prerequisites beyond arithmetic.

## Contents

- **Chapter 1 — Sets**: set notation, membership, subsets, union, intersection, and complement
- **Chapter 2 — Functions**: definition, domain, codomain, composition, and inverses
- **Chapter 3 — Linear Equations**: one variable, systems of two equations, and geometric interpretation
