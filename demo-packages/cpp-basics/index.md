# C++ Basics

Welcome to **C++ Basics** — a ground-up introduction to the C++ programming language.

C++ is a compiled, statically typed, multi-paradigm language offering fine-grained control over hardware resources. It is widely used in systems programming, game development, embedded systems, and performance-critical applications.
