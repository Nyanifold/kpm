# python-basics

A knowledge package covering the fundamentals of Python programming. Suitable for complete beginners with no prior programming experience.

## Contents

- **Chapter 1 — Syntax & Variables**: data types, variables, expressions, and basic I/O
- **Chapter 2 — Functions**: defining functions, parameters, return values, and scope

## Languages

This package is available in English (`en`) and Simplified Chinese (`zh-CN`).
